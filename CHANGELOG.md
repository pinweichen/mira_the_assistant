# Changelog

## v1.0.0 — 2026-03-29

### Added
- macOS installer (`setup.sh`) — 8-phase idempotent setup
- Conservative uninstaller (`uninstall.sh`)
- Voice setup: macOS `say` voices + optional VibeVoice neural TTS
- Discord bot integration (optional)
- Google Calendar and email access via gws plugin
- Claude Code plugin auto-installation (discord, remember, claude-md-management, hookify, superpowers, gws)
- whisper-cpp speech-to-text setup
- Template system with `{{PLACEHOLDER}}` variable substitution
- macOS `.app` launcher bundle generation
- Workspace scaffolding (CLAUDE.md, tasks, tracker, templates)
- User guide documentation

### Initial Release
First public release targeting macOS (Apple Silicon and Intel). Windows support planned for v2.
