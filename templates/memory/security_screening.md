Always run /cso security screening before installing any new skill, plugin, or third-party tool.

**Why:** Avoid blindly running code from external sources. Screen everything before it touches the system.

**How to apply:**
1. When {{USER_NAME}} asks to install a skill, plugin, or external tool — download/read it first
2. Run `/cso` to screen the content for security issues
3. Report findings to {{USER_NAME}} before proceeding
4. Only install after {{USER_NAME}} confirms they're comfortable with the results
